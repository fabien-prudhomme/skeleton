### Tests

Run all tests:
- PHPCs
- PHPStan
- PHPUnit
 
```bash
make tests
```

Run PHPCs tests:

```bash
make phpcs
```

Run PHPStan tests:

```bash
make phpstan
```

Run PHPUnit tests:

```bash
make phpunit
```
